# Wedding Website

Personal web application created for our wedding.

**Technologies Used:** React | Django | Python | JavaScript | HTML | CSS

---

## Features

- Welcome page  
- Wedding preparation updates  
- Direct messaging from guests  
- Recommended stays for guests  
- Picture gallery browsing  
- Countdown timer until the wedding date

---

## Usage

1. **Clone the repository:**
```bash
git clone https://github.com/FilipBosilkov/wedding.git
cd wedding
```

2. **Install Python dependencies:**
```bash
pip install -r requirements.txt
```

3. **Run Django server:**
```bash
python manage.py runserver
```

4. **Access the site:**
Open your browser at `http://127.0.0.1:8000/`
